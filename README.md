# plain-ui
A simple react tailwind UI component library
> This Project is under development

### 1. Install

`npm i @reactizard/plain-ui`

### 2. Usage

```jsx
import React from 'react';
import { Button } from '@reactizard/plain-ui';

function App() {
  return (
    <Button color="purple" size="sm">
      Button
    </Button>
  )
}
```
